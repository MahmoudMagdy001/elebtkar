## 🛠 التقنيات المستخدمة | Tech Stack

- **Frontend**: HTML5, Vanilla CSS3, Modern JavaScript (ES6+).
- **Icons**: [Phosphor Icons](https://phosphoricons.com/).
- **Backend/Database**: [Supabase](https://supabase.com/).
- **Deployment**: [Vercel](https://vercel.com/) / Hostinger (PHP compatible).
- **Fonts**: [Google Fonts (Almarai)](https://fonts.google.com/specimen/Almarai).

## 📂 هيكلة المشروع | Project Structure

```text
elebtkar/
├── assets/          # Shared CSS, images, docs, and scripts
├── components/      # Reusable HTML/JS components
├── pages/           # Feature-specific pages (Admin, Blog, Home, Post)
├── index.html       # Main landing page
├── vercel.json      # Vercel routing configuration
└── .htaccess        # Apache/Hostinger routing configuration
```

## 💻 البدء بالعمل | Getting Started

### المتطلبات | Prerequisites
- متصفح ويب حديث (Modern Web Browser).
- مفاتيح Supabase (توجد في `assets/js/config.js`).

### التشغيل المحلي | Local Setup
1. قم بتحميل المستودع (Clone the repository).
2. افتح ملف `index.html` مباشرة أو استخدم Lite Server/VS Code Live Server.
